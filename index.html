Análise da Dinâmica do Pêndulo de Ondas¶

Este projeto explora a modelagem matemática e a validação experimental de um sistema de pêndulos simples sob a influência de forças de arrasto. O objetivo é compreender como variações no comprimento do fio impactam o período de oscilação e o alcance vertical, comparando simulações numéricas com dados reais obtidos via rastreamento de vídeo (Tracker).

2. Modelagem Física e Simulação¶

O modelo considera a segunda lei de Newton aplicada ao pêndulo, incluindo a força peso, a tração no fio e a força de arrasto do ar ($F_a = \frac{1}{2} \rho A C_d v^2$).

params = get_simulation_parameters()

# Configuração da simulação base
L = 0.24  # Comprimento do fio (m)

# Executa a simulação
results = sim.simulate(L, params['m'], params['r'], initial_state, t_lista)
x_lista, y_lista = results[:, 0], results[:, 1]

Nesta seção, simulamos uma série de pêndulos com diferentes comprimentos para observar a formação do padrão de ondas.

periodos = []

5. Conclusões¶

A análise desenvolvida neste projeto permitiu integrar a modelagem teórica da mecânica clássica com dados experimentais reais, resultando em uma compreensão aprofundada sobre a dinâmica de oscilação e perda de energia do sistema. A partir dos resultados, destacam-se os seguintes pontos:

  1. Validação do Modelo com Força de Arrasto: A inclusão da força de arrasto aerodinâmico ($F_a \propto v^2$) na resolução das Equações Diferenciais Ordinárias (ODE) mostrou-se fundamental. Ao cruzar os dados da simulação com os pontos coletados pelo software Tracker, ficou evidente que um modelo ideal (sem atrito) falharia em prever o decaimento da amplitude. O modelo numérico desenvolvido obteve altíssima aderência à realidade.

  2. Dependência Paramétrica para o Pêndulo de Ondas: A varredura de parâmetros comprovou a relação não-linear entre o comprimento do fio ($L$) e o período de oscilação ($T$). Essa precisão paramétrica é o "coração" do fenômeno do Pêndulo de Ondas, pois permite o cálculo exato do comprimento que cada pêndulo deve ter para que o conjunto entre em sincronia e produza os padrões de onda desejados.

  3. Escalabilidade Computacional: A utilização de Programação Orientada a Objetos para isolar a lógica física no módulo physics.py garante uma arquitetura de software escalável. Com essa base estruturada e uso eficiente do numpy, a simulação pode ser expandida facilmente para dezenas de pêndulos interativos sem perda de performance.

Em suma, o projeto atesta que a combinação de integração numérica avançada (via scipy) com a instrumentação de dados visuais é uma abordagem altamente robusta para a validação de sistemas dinâmicos e desenvolvimento de simulações físicas precisas.
